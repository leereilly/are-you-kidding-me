Meat Makes Us Go? A physics puzzler made in 48 hours for the 2013 Global Game Jam.

Click floor tiles to make waves that push the ball along. Multiple clicks produce intersecting waves that will accumulate and make taller waves. Leap over chasms, jump walls, and walk along perilous pathways.

Design and code by @AlBakerKY. Additional code by @mildmojo. Sound by Mr. Mathieu Keith. Code available.
